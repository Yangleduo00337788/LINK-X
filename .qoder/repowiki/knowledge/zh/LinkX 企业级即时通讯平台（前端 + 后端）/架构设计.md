根目录作为多子工程聚合层，按 `linkx-client/`（Electron 主进程 + Vue3 渲染进程，通过 preload + contextBridge 暴露 `window.electronAPI`）与 `linkx-server/`（Spring Boot 单体，MySQL/Redis/MinIO 由 docker-compose 编排）划分前后端边界；两者无共享代码，仅以环境变量 `VITE_API_BASE_URL` 约定 REST 前缀 `/api`，并通过 JWT Access/Refresh 双 Token + Redis 吊销实现认证契约。根 README 统一描述构建命令、状态管理（Pinia）、CSS Design Tokens、安全存储等跨模块约定，是唯一的入口文档。
