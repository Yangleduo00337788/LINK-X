### 真实会话 / Mock 会话
- 定义：LinkX 中区分两种聊天会话：isReal=true 的真实会话走后端 HTTP+WebSocket 链路（拉取历史、发送消息经 Netty 转发），isReal=false 的演示会话仅使用本地 mockData.ts 数据，不触发任何网络请求。app.ts 中通过 session.isReal 字段分支处理。
- 别名：real session、demo session

### 双 Token
- 定义：认证体系采用 Access Token（2 小时有效期）+ Refresh Token（7 天有效期）组合。Access Token 随每个请求携带 Bearer 头，Refresh Token 存 Redis 支持服务端吊销。前端 axios 拦截器遇 401 自动用 Refresh Token 换票并重试原请求，避免并发刷新冲突。
- 别名：access token、refresh token

### 乐观更新 + ack 替换
- 定义：IM 消息发送流程：客户端先插入一条 temp- 前缀临时消息立即渲染，随后通过 WebSocket 发送 action=send；服务端确认后回推 action=ack，客户端根据 clientMsgId 将临时消息替换为服务端确认后的正式消息。
- 别名：temp message、message ack

### 群元数据
- 定义：群聊扩展能力的本地状态集合，包括群公告、群精华、群成员、群备注、群文件、群相册六个子域，按 sessionId 懒加载默认演示数据，后续需对接后端接口实现持久化。
- 别名：group meta、group extension

### 好友通知
- 定义：好友相关系统通知，分为 incoming（他人请求加我）和 outgoing（我邀请他人）两个方向，状态包括等待验证、已同意、已拒绝。前端从后端 friendApi.listIncomingRequests/listOutgoingRequests 拉取后统一映射到 FriendNotification 列表展示。
- 别名：friend request notification

### 内嵌应用
- 定义：通过 Electron WebView 内嵌第三方网页应用的机制，当前预置网易云音乐和抖音两类应用，媒体播放状态由 mediaPlaybackStore 解析页面标题动态提取当前播放曲目信息并在状态栏展示。
- 别名：embedded app、webview app
