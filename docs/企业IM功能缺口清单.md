## 一、核心能力（已交付）

### 1. 消息可靠性闭环
- 发送中 / 成功 / 失败状态
- 离线补偿与重连后 sync
- `clientMsgId` 幂等去重
- 失败自动退避重试（最多 2 次）+ 手动重试
- 历史顺序：雪花 `id` 升序 + `beforeMessageId` 游标分页

### 2. 已读 / 未读 / 送达回执
- 已送达（deliveryReceipt）
- 已读 / 群聊已读人数
- 未读数与角标联动

### 3. 消息编辑、撤回、转发、引用
- 撤回 / 编辑 / 转发 / 引用回复
- 搜索与 @我 可定位到气泡并短暂高亮

### 4. 全文搜索与会话搜索
- 关键词搜消息（时间范围 + highlight）
- 综合搜索：用户 / 群 / 消息（可定位气泡）
- 聊天记录页日期筛选 + 跳转定位

### 5. 多端同步与设备管理
- 多设备登录、设备列表、踢下线（吊销 token + 断 WS）
- 已读状态多端归零

---

## 二、企业化增强（已交付）

### 6. 会话管理
- 置顶 / 免打扰 / 草稿
- **重要会话高亮**（独立于置顶；右键标记；列表高亮）
- 排序：重要 > 置顶 > 时间

### 7. 群管理
- owner / admin / member
- 禁言 / 入群审核 / 公告已读 / 批量管理 / 邀请策略

### 8. 通知
- 桌面端**本地** Notification（刻意不做 FCM）
- 聚合（type + relatedId，24h）
- 免打扰时段 + 通道开关（云端偏好）
- 业务 / 系统分轨

### 9. 文件与媒体
- 大文件分片、断点续传、秒传、过期链接自愈、预览

### 10. 安全与审计
- 验证码 + 登录/刷新限流
- 黑名单、敏感词、操作审计、数据脱敏

---

## 三、高阶能力

### 11. 超大群与性能（已交付主体）
- 分片广播、风暴检测落库、热消息 Redis、id 游标索引

### 12. 音视频
- ICE restart、弱网降码、设备切换、权限拒绝降级语音
- 会议房：管理房 + 本地预览（**非完整 mesh 多人媒体面**）

### 13. 合规
- 留存任务、导出、清除、合规审计

---

## 四、明确不在当前交付范围（可选后续）

| 项 | 说明 |
|----|------|
| 完整 mesh 多人会议 | 需独立媒体面工程 |
| FCM / 服务端推送 | 刻意收口为本地通知 |
| 系统级大群压测工具 | 现有 12 人扇出冒烟 |
| 真实双端 WebRTC + 系统丢包 | 现有策略单测 + CDP/Fake |
| 前端覆盖率大幅抬升 | 门禁为最低门槛（见测试文档） |
