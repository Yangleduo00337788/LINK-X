# LinkX 企业 IM 功能缺口清单

## 一、必须补齐的核心能力

### 1. 消息可靠性闭环
企业 IM 的核心不是“能发消息”，而是“消息可靠送达并可追踪”。

缺口包括：
- 发送中 / 成功 / 失败状态
- 离线消息补偿
- 重连后消息同步
- 消息幂等去重
- 消息顺序保障
- 失败重试机制

### 2. 已读 / 未读 / 送达回执
成熟 IM 通常至少具备：
- 已送达
- 已读
- 群聊已读人数统计
- 单聊 / 群聊不同回执策略
- 未读数与角标联动

### 3. 消息编辑、撤回、转发
企业场景很常见：
- 撤回消息
- 编辑消息
- 转发消息
- 引用回复链
- 原消息定位与跳转

### 4. 全文搜索与会话搜索
常见检索能力包括：
- 按关键词搜消息
- 按联系人 / 群组 / 文件搜索
- 时间范围筛选
- 结果高亮
- 定位到具体消息

### 5. 多端同步与设备管理
成熟企业 IM 通常需要：
- 多设备同时登录
- 在线设备列表
- 踢下线
- 设备状态同步
- 已读状态同步

## 二、建议增强的企业化能力

### 6. 会话管理增强
建议补充：
- 会话置顶
- 免打扰 / 静音
- 草稿保存
- 会话排序策略
- 重要会话高亮

### 7. 群管理体系增强
企业群需要更细的权限体系：
- 群主 / 管理员 / 普通成员
- 禁言 / 解禁
- 入群审核
- 群公告已读统计
- 群成员批量管理
- 群聊邀请策略

### 8. 通知系统增强
建议完善：
- 桌面通知
- 系统通知
- 通知聚合
- 通知策略控制
- 业务通知与系统通知分离

### 9. 文件与媒体能力增强
当前已有文件体系，但企业级还应支持：
- 大文件上传
- 断点续传
- 秒传 / 去重
- 预览能力
- 权限控制
- 过期链接
- 审计留痕

### 10. 安全与审计
企业环境必备：
- 登录风控
- 黑名单
- 敏感词过滤
- 操作审计
- 管理员追踪
- 限流 / 反刷
- 数据脱敏

## 三、高阶能力

### 11. 超大群与性能优化
如果未来要进一步对标成熟 IM，需要考虑：
- 大群消息分发
- 分片广播
- 消息风暴控制
- 历史消息分页优化
- 缓存与索引策略

### 12. 音视频通话完善
你已有通话能力，但企业级还要看：
- 断线重连
- 弱网处理
- 多人会议
- 设备切换
- 权限异常处理

### 13. 合规与数据治理
企业场景常要求：
- 数据留存策略
- 数据导出
- 删除策略
- 合规审计
- 敏感信息治理
